# 概要
- Node.js ：CSVファイルを読み、パースして出力する。

# 使用技術
- node
  > node --version

  > v16.17.0

- npm
  > npm --version

  > 8.18.0

## 使用ライブラリ
1. [ CSV parser ] for Node.js and the web
  - CSVファイルを読み込んでパースする

# 手順
1. npm初期化
    - > npm init -y
1. [CSV parser]
    - > npm install csv-parse --save 

# 実行
> node index.js
